# Chapter [3]: 제네릭 활용

## item[29]: 컬렉션을 반환하기보다 이터레이터를 반환하는 것이 낫다

- 이터레이터(Iterator)
  - 데이터 컬렉션을 순회할 수 있는 객체. yield return 키워드를 사용하여 구현할 수 있다.
- yield return
  - 이터레이터를 구현하는 데 사용되는 C#의 키워드로, 메서드가 반환할 값을 하나씩 순차적으로 생성. 메서드가 호출될 때마다 중단된 위치에서 계속 실행됨.
